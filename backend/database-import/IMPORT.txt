Import into MongoDB database "road-accident-system" (create empty DB in Compass or use mongoimport).

Commands (MongoDB Database Tools):

mongoimport --uri mongodb://localhost:27017/road-accident-system --collection users --drop --file users.json --jsonArray

mongoimport --uri mongodb://localhost:27017/road-accident-system --collection hotspots --drop --file hotspots.json --jsonArray
